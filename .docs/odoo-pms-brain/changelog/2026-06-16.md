# Changelog - 2026-06-16

## [Added] PMS Module Initialization

- Initialized the new custom Odoo module `pms` focused on Vacation Rentals.
- Created `pms.property` model representing the Vacation Rental entity.
  - Linked to `product.template` using Odoo delegation inheritance (`_inherits`).
  - Mapped properties and metadata to Schema.org's `VacationRental` standard.
- Scaffolded module layout (`models/`, `views/`, `security/`, manifest, etc.).
- Configured views, security files, and top-level menus for "Property Management".

## [Added] Booking and Sales Integration

- Created the `pms.booking` model to manage vacation rental guest reservations.
- Integrated bookings with Odoo Sales (`sale.order` and `sale.order.line`).
- Added automatic state synchronization from Sales Orders to bookings.
- Designed python check-in/check-out lifecycle transitions.
- Implemented overlap constraint validation to prevent double bookings.
- Exposed booking tree/form views, action flows, and menu navigation.
