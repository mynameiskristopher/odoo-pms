# Changelog - 2026-06-26

## [Added] Hierarchical Zones and Customizable Zone Types

- Created `pms.zone.type` model to allow customizable classifications of zones (e.g. Region, Building, Floor).
- Created `pms.zone` model to represent hierarchical zones, utilizing Odoo's native `_parent_store` and `complete_name` patterns for nested paths.
- Added `zone_id` Many2one field to `pms.property` model to link properties to their corresponding parent zone.
- Added views and menus:
  - Created list, form, and search views for `pms.zone` and `pms.zone.type` in `views/pms_zone_views.xml`.
  - Added a new top-level "Configuration" category to the main PMS menu.
  - Placed "Zone Types" under Configuration and "Zones" under the Properties menu.
- Updated `pms.property` views:
  - Added `zone_id` field to the property list and form view (under Property Specifications).
  - Added search by zone and group-by zone filters to the property search view.
- Added access rules for the new models in `ir.model.access.csv` to allow all portal/standard users full read/write access.
